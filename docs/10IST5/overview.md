### Introduce the Unit

- Students will review Software Development and Programming:
	- Understand algorithms
        - Use design techniques (flowchart and pseudocode) to plan algorithms
        - Sequence
        - Selection
        - Repetition
        - Convert algorithms into Python
	- Python
	    - Data types
        - Parts of Program
            - Control structures
            - Variables
            - Operators
            - Assignment
            - Built-in functions
            - Data structures
- Students will review Animations
    - Animation Techniques
        - Tween
        - Motion tween
        - Keyframe and Parenting
        - Workspace
        - Vector and Bitmap graphics
        - Rigging
        - Virtual Camera
        - Onion Skinning
